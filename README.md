# 项目说明

这是一个基于Django官方文档的投票系统。

## 快速开始
启动服务：`python manage.py runserver`

## 目录结构
```
djgproj/
├── manage.py
├── djgproj/
│   ├── __init__.py
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
└── polls/
    ├── __init__.py
    ├── admin.py
    ├── apps.py
    ├── migrations/
    │   └── __init__.py
    ├── models.py
    ├── tests.py
    └── views.py
```
